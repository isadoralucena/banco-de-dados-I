## Descrição

Este projeto foi desenvolvido no âmbito da disciplina Banco de Dados I — UFCG, período 2025.2, e tem como objetivo a **modelagem e implementação de um banco de dados** para uma plataforma de streaming fictícia denominada **StreamVerse**.

Ao longo do projeto, são abordados conceitos fundamentais de modelagem de dados, normalização, criação do banco de dados e desenvolvimento de consultas e objetos de banco utilizando SQL e PL/SQL.

## Fases do Projeto

- **Fase 1 – Modelagem Conceitual (MER):**  
  Identificação das entidades, atributos e relacionamentos que compõem o domínio da plataforma de streaming.

- **Fase 2 – Modelo Relacional:**  
  Conversão do Modelo Entidade-Relacionamento (MER) para o modelo relacional, com aplicação de técnicas de normalização e criação da estrutura do banco de dados.

- **Fase 3 – Consultas SQL (DML):**  
  Desenvolvimento de consultas SQL utilizando **SELECT**, **JOINs**, **subqueries**, **agrupamentos** e **filtros**, visando à extração e análise dos dados.

- **Fase 4 – Objetos de Banco de Dados:**  
  Criação de **Stored Procedures**, **User-Defined Functions**, **Triggers** e **Views**, com foco em **integridade referencial**, **consistência dos dados**, **regras de negócio** e **automação de processos**.
